Coding

Your avatar will move according to your programmed instructions. Once the game starts, you will have no influence on the behavior of your avatar. It will execute your instructions at each round to try to find a way out of the maze, like a robot.

This means that it is extremely important to write good code that will execute when the game starts. To write code, head to the Edit Code screen.

Open a panel by clicking on it:

↓

You can then view a selection of blocks from this panel:

↓

Select the block you want and drag it to the working area:

↓

You can also select from a list of pre-defined algorithms, using the load option:


↓

The selected algorithm will be loaded into the work area:

↓

You can zoom in and out, center the screen to your code or delete blocks by dragging them on the bin:

↓

When you are done, save your code:


Make sure you use a valid name (only letters and numbers). When the code is saved, you can load it just like you loaded the sample code.

↓

You can clear your workspace using the clear button:

↓

↓
